Gym CRM Spring MVC project

This is the main page of the Gym CRM Java Spring MVC project.

Technologies used:

  • Spring Framework (Spring Core, Spring MVC, Spring Data JPA)
  • Hibernate
  • PostgreSQL
  • JUnit5, Mockito
  • Logback for logging
  • HTML, Bootstrap, Thymeleaf for front-end
  • Gradle
  • JaCoCo plugin for code coverage analysis

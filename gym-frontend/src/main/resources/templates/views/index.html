Gym CRM Spring MVC project

This is the main page of the Gym CRM Java Spring MVC project.

To access any functionality except adding a trainee or adding a trainer your need to log in.

Technologies used:

  • Spring Framework (Spring Boot, Spring Core, Spring MVC, Spring Data JPA, Spring Security)
  • Hibernate
  • PostgreSQL and H2 databases
  • JUnit5, Mockito
  • Logback for logging
  • HTML, CSS, Bootstrap, Thymeleaf for front-end
  • Gradle
  • JaCoCo plugin for code coverage analysis
  • Swagger
